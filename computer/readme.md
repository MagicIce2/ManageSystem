### [计算机社团人员管理系统](http://211.67.160.21/computer/jsp/index)

##### 开发规范

- 参考阿里巴巴 Java 开发手册

本项目基于SSH（Struts2、Spring、Hibernate），使用 maven 项目构建工具

##### 项目环境

jdk：1.8

IDE：idea2018.2.4

Web服务器：Tomcat9.0（若需要进行负载均衡，可使用nginx）

maven：apache-maven-3.6.1

项目打包：mvn package

sql文件：student.sql



为防止office版本问题，导致编码不对，请查看pdf文档。
